# SubjectLine Generator

🚀 Just built an AI-Powered Email Subject Line Generator

Created a full-stack application that generates high-converting, spam-safe, mobile-optimized email subject lines using FastAPI + Gemini API + LangChain.

✨ Features:
 • AI-generated subject lines tailored by tone, audience & industry
 • Spam detection + scoring system (0–100)
 • Mobile length optimization (15–60 chars)
 • Retry & validation logic for reliable outputs
 • Clean, responsive frontend with real-time results

Built with Python, FastAPI, Pydantic, HTML, CSS & JavaScript.
Excited to keep building practical AI tools that solve real marketing problems .
